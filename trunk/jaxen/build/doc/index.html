SAXPath: Simple API for XPath

The SAXPath project is a Simple API for XPath. SAXPath is analogous to SAX in that the API abstracts away the details of parsing and provides a simple event based callback interface.

Getting Started

You could try browsing the FAQ or the online JavaDoc.

Contributors are welcome to join this project. Once you've browsed the FAQ you could try sending an email to one of the mailing lists below or check out the project page.
